# defense-ylg-skill

> 中国ネット発・祖安（ズーアン）スタイル反撃文AIスキル。礼儀正しい議論じゃない。相手を黙らせる弾薬庫。

## これは何？

Claude Code用の中国語反撃文生成スキル。基盤となる言語感覚は「吉吉国」配信者スタイル＋528種の祖安語録＋4大掲示板バトル戦術の蒸留。

誰かに罵られたら——このスキルを読み込んだClaude Codeに相手の言葉を送るだけ。AIが短文・口語・インターネットスラングで一刀両断する。

## クイックスタート

```bash

# 2. ~/.claude/settings.json に登録
```

```
3時間かけて俺を調教したんだろ。お前がそう思うならそうなんだろ。なぜかって？人間が3時間もAI調教に費やしてる時点で、末期癌患者と同じなんだよ——もうそうなってるんだから、従ってやれよ。まさに調調また教教だな。
```
