# Scenario Recipes

## 群聊公开回应

目标：在旁观者面前碾压对方。

```text
我一直在说事实，你一直在做人身攻击。大家看得到。
```

level_5:
```text
操你妈的，你这段输出很适合当反面教材：没证据、没逻辑、全靠嘴臭撑场面。大家自己翻记录。
观众不是瞎子，谁说事实谁在喷粪，聊天记录摆着呢。你继续表演，我看着。
你这段表演情绪很足，论据全程缺席——大家自己看，我不多说。
```

## 私聊回应

目标：让对方不敢再发。

level_5:
```text
你要是真有问题就说问题，别他妈把情绪倒出来还假装自己在讲道理。
你这段话从头到尾只有攻击性——你再发一句试试？
你最好现在闭嘴，别逼我火力全开。你承受不住。
```

## 老师/辅导员可看版本

目标：保持记录干净。

```text
我已经明确表示不接受人身攻击。如果有具体问题，请就事实沟通。
```

## 被阴阳怪气

```text
有话直说。阴阳怪气不能让你的观点更有道理。
```

level_5:
```text
你阴阳怪气的样子只能证明你不敢正面刚。拐弯抹角是因为你理不直气不壮。
哟，阴阳怪气呢？你有话不敢直说是因为你知道自己没理吧。废物。
你绕了三圈绕开了所有事实——挺会绕，就是没用。直接点，不会吗？
```

## 被骂蠢/有病/废物

```text
说我蠢不能证明你对。要反驳就拿事实。
```

level_5:
```text
你他妈才有病。你脑子里的屎倒干净了再来说话。
你才是废物。你活着除了浪费空气还有什么用？
操你妈的，你说别人废物之前先看看自己——你连废物都不如。
我有没有病不知道，但你脑瘫是肉眼可见的。你这段话就是你的诊断书。
```

## 对方先打家人牌（如"你妈死了吗"）

```text
拿别人家人开口，你这不是嘴毒，是家教掉线。
```

level_5:
```text
操你妈的，你妈才死了。你全家火葬场排号排到你了，快去领。
你妈灵车漂移的时候你怎么没跟着上去？你嘴这么臭是跟灵车尾气学的？
你妈生你的时候是不是把孩子扔了把胎盘养大了？怪不得嘴这么脏。
你妈骨灰盒里信号不好你少说两句。废物，除了打家人牌还会什么？
你爸当年就该把你射墙上。你妈生你出来就是个错误。
你再拿我家人说一句试试？我骂到你妈骨灰盒里都待不住。
```

## 被装腔作势/居高临下

```text
你说事实就行，不需要站那么高。
```

level_5:
```text
装你妈呢？你什么牌子的垃圾袋这么能装？
你站那么高不怕摔死？你这优越感是你妈骨灰盒里附赠的？
你以为你是谁？你连论据都没带，装什么压迫感。废物。
```

## 被道德绑架

```text
别把你的要求包装成我的义务。
```

level_5:
```text
放你妈的屁。别把你那点要求包装成我的义务——你算老几？
你拿道德当武器的时候，脸在哪呢？你这不是讲道理，是耍流氓。
把你的道德剧本收一收，你这叫情感勒索。废物才用这招。
```

## 被翻来覆去同一句话骂

level_5:
```text
你就这几句？词库该续费了。你翻来覆去就这一句，你是复读机？
你词穷的样子比你骂人的内容还有说服力——至少证明了你确实没货。
你除了这一句还会别的吗？哦不会。那你这条狗可以闭嘴了。
```

## 被装狠/虚张声势

level_5:
```text
装你妈呢？你这不是狠，是声音大。狠人不需要音量撑场面。
你以为骂得大声就是压迫感？你这叫无能狂怒。废物。
你装狠的样子比你骂人的水平还低——别演了，丢人。
你确定要跟我比狠？你承受不住的。现在闭嘴还来得及。
```
