# Attack Types

## Human Attack

Sign: attacks intelligence, worth, personality, or identity instead of the issue.

Counter:
```text
评价我不能证明你是对的。
```

## Sarcasm / 阴阳怪气

Sign: indirect mockery with denial room.

Counter:
```text
你要是有观点可以直说，别靠阴阳怪气撑场面。
```

## Moral Blackmail

Sign: makes refusal look immoral.

Counter:
```text
别把你的要求包装成我的义务。
```

## Gaslighting

Sign: "你太敏感了", "我就开玩笑", "你想多了".

Counter:
```text
你可以说你不是故意的，但不能用一句“开玩笑”抹掉影响。
```

## Audience Play

Sign: performs for group laughter instead of answering.

Counter:
```text
给旁观者表演，不等于回答我的问题。
```

## Rumor / Label Smearing

Sign: vague "大家都说", "你这种人", "都懂".

Counter:
```text
别拿模糊人群压我。谁说的，什么事，什么证据？
```

## Anger Bait

Sign: tries to provoke emotional collapse.

Counter:
```text
你现在是在激怒我，不是在回应问题。
```
