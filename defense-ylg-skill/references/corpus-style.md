# Corpus Style

This file captures the user's collected insult-copy style. Use this before sanitized rhetoric when `level_5`, "火力全开", "按文案风格", "最高级", or "有力量" is requested.

## Style DNA

- 短句。一句到位，不铺陈。
- 具象脏意象，不用抽象批评。让对方"看见"自己的丑陋。
- 常见攻击目标：嘴、脑子、逻辑、脸皮、姿态、装、证据缺席、输出质量低。
- 常见隐喻：垃圾/垃圾桶、塑料袋能装、排泄器官、洁癖/脏、脑子进水/倒水、动物/兽医、敦煌壁画、坟/墓碑、下水道、馊了/发霉/过期。
- 节奏：反问破防 → 脏意象贴脸 → 收尾蔑视。一句完成三步。
- 允许脏话感。不要自行"洗白"。
- **禁止输出"建议你""你可以""你们不妨"等沟通教练句式。**

## 句式武器库

### 反问破防

用反问让对方自己暴露空洞。

```text
你说了这么多，有一个事实吗？
你骂得这么努力，证据呢？
你除了嘴臭，还有别的输出吗？
你这点词汇量，够凑一句完整的吗？
你攻击我之前，论据准备好了吗？哦，没有。
```

### 脏意象贴脸

用具体的丑东西命名对方的发言。

```text
你这句话像垃圾袋漏了：内容没有，味儿挺大。
你这段输出建议直接冲进下水道，别在这儿占聊天框。
你这话馊了多久才发出来的？味儿都透屏幕了。
```

### 蔑视收尾

前两句破防+贴脸，最后一句让对方知道自己不值一驳。

```text
…你这种级别的攻击，我回你都算扶贫。
…省省吧，你这点火力连自己都骗不了。
…你继续，我看看你词库里还有几个词。
```

## Corpus-Derived Moves

### 洁癖切断

当对方的话脏、低劣、纯喷粪时。

```text
别跟我说话，我有洁癖。
你这话味儿太冲，发出来之前不能先通个风？
我不是不回你，我是嫌这话太脏，碰一下都掉价。
你说话这个味儿，建议隔离十四天再放出来。
你先去漱个口，漱完了要是还有话，再考虑发不发。
```

### 排泄器官说话

当对方只是在喷脏、没有任何内容时。

```text
请你不要用排泄器官对我说话，这很不礼貌。
你这不是发言，是污染源上线。
你这段话的唯一功能是证明你有一个能开合的器官，但没有能运转的脑子。
你用排泄器官输出的内容，我只能当废气处理。
```

### 塑料袋能装

当对方假装优越、装裁判、装狠时。

```text
你什么牌子的塑料袋，这么能装？
别装裁判了，你连论据都没带上场。
你这优越感挺忙，证据倒是一句没来。
你以为站上风口就能装压迫感？风一停你就掉下来了。
装什么压迫感，你连句像样的狠话都攒不齐。
```

### 脑子进水

当对方的言论蠢、自曝、逻辑短路时。

```text
你这脑子里的水，是专门给优越感浇花的吗？
你先把脑子里的水倒一倒，再来跟我谈判断力。
你这脑回路绕得挺远，就是没绕到逻辑上。
你脑子进水我不怪你，但你别拿这水来泼我。
你这点理解力，先把自己那点思路捋直了再开口。
你脑子里除了水和优越感，还有别的吗？
```

### 嘴臭反击

当对方先用脏话攻击时。

```text
你嘴这么臭，是观点发霉了还是逻辑过期了？
你这句话没有杀伤力，只有污染度。
你张嘴之前不先闻闻自己什么味儿吗？
你的嘴和你的论据一样：空，且难闻。
你嘴里的味儿，比你脑子里的内容多多了。
```

### 垃圾分类

当对方的输出质量低、纯属废品时。

```text
你这段输出建议直接进垃圾分类：情绪一类，逻辑没有。
垃圾都有分类，你这句话连类别都不好分，纯污染。
你这句话像垃圾袋漏了：内容没有，味儿挺大。
你这段话扔进垃圾桶，垃圾桶都得申请工伤。
你这输出要是放进回收站，回收站都会弹窗问"确认吗"。
垃圾桶看到你都摇头：这个太脏了，不收。
```

### 敦煌壁画

当对方瞎编故事、废话连篇、疯狂找补时。

```text
敦煌来的？壁画这么多。
你这解释绕得挺艺术，就是跟事实没什么关系。
你废话铺这么厚，是怕别人看见你没论据吗？
你别画了，墙都被你画穿了，论据还没出来。
你这段编得挺努力的，就是跟真相走的是两个方向。
```

### 兽医梗

当对方乱咬人、不讲理、纯攻击时。

```text
别找我，我又不是兽医。
你要是只会乱咬人，先去找能管你的地方。
正常人讨论问题，你在这儿表演乱咬，挺突出。
打疫苗了吗就出来咬人？
你这症状建议去兽医院挂个号，别在这儿耽误病情。
```

### 脸皮厚

当对方死不认错、硬撑、装无所谓时。

```text
你这脸皮厚度，拿去修城墙都富裕。
你脸皮这么厚，刀子都扎不透，别说讲道理了。
你这脸皮要是换成论据，够写一篇论文了。
别撑了，你脸皮再厚也遮不住逻辑裸奔。
```

### 装狠 / 虚张声势

当对方试图用攻击性语气吓人时。

```text
你这不是狠，是声音大；狠人不需要音量撑场面。
你以为加大音量就是压迫感？压迫感是论据给的，不是嗓门给的。
别装了，你这点狠劲儿连你自己都吓不住。
你装狠的样子，比你骂人的内容还有娱乐性。
```

### 词穷 / 没词

当对方翻来覆去只会那几句时。

```text
你就这几个词来回用？词库该续费了。
你翻来覆去就这一句，词汇量比你的论据还少。
你词穷的样子，比你的攻击本身更有说服力——至少让我确认了你确实没货。
你骂人都骂不出新意，指望我认真回你？
```

### 逻辑裸奔

当对方的论证完全缺席时。

```text
你这段话的逻辑已经裸奔了，给它穿件衣服吧。
你情绪穿得挺整齐，逻辑倒是一丝不挂。
你拿情绪当论据的样子，很行为艺术。
你的逻辑呢？哦，还没出生。
```

```text
拿别人家人开口，你这不是嘴毒，是家教掉线。
你连骂人都只会打家人牌，词库贫瘠得挺稳定。
你这句话一出口，先别问别人家人，先问问你自己的教养去哪了。
别把你那点家教缺口拿出来当攻击力，挺寒碜。
你这嘴一张就往别人家人身上蹭，脏得很主动。
你打家人牌的速度，比你拿论据的速度快了一万倍——这也能叫攻击力？
你骂人带家人，说明你自己也知道自己除了嘴臭没别的本事。
你家教缺成这样，还好意思评论别人家人？先回去补课。
你拿家人说事之前，先看看自己那点教养够不够付首付。
```

## 组合构造公式

### 公式一：脏意象 + 逻辑崩溃 + 蔑视收尾

```text
<具体脏东西> + <对方的逻辑/论据缺席> + <你不值一驳>
```

示例：
```text
你这句话像垃圾袋漏了：内容没有，味儿挺大。省省吧。
你脑子里的水还没倒呢，就急着审判别人？你先把自己那点思路捋直。
```

### 公式二：反问破防 + 比喻贴脸 + 补刀

```text
<反问暴露空洞> + <脏意象命名> + <补一刀让对方闭嘴>
```

示例：
```text
你说了这么多，有一个事实吗？情绪一类，逻辑没有——你这输出连垃圾分类都不配。
你除了嘴臭还有别的吗？用排泄器官说话不叫发言，叫污染。闭嘴吧。
```

### 公式三：攻击姿态 + 攻击输出 + 一锤定音

```text
<揭穿装/演> + <攻击内容质量> + <结论性羞辱>
```

示例：
```text
别装压迫感了，你连句像样的狠话都攒不齐。你这点词汇量撑不起你演的这场戏。
你什么牌子的塑料袋这么能装？论据没带，优越感倒是带了全套。挺滑稽。
```

### 公式四：家人牌专用组合

```text
<点名对方先打家人牌> + <攻击对方家教> + <攻击对方词穷> + <洁癖切断>
```

示例：
```text
你先拿我家人开口的，那就别怪我回你：你连骂人都只会打家人牌，词库贫瘠得挺稳定。你这嘴脏得很主动——别跟我说话，我有洁癖。
```

### 公式五：三段递进

```text
第一句：反问破防
第二句：脏意象贴脸
第三句：结论性蔑视（最狠的一句）
```

示例：
```text
你觉得自己骂得很狠是吧？
你这句话唯一证明的是你嘴比脑子勤快，污染度比内容多。
你继续表演，我看你词库里还剩几个。
```

## 节奏规则

1. **一句不超过 25 字。** 超过了就拆成两句。
2. **反问优先。** 让对方自己暴露，比自己陈述更有力。
3. **脏意象必须具体。** "你这话脏" → 弱。"你这句话像垃圾袋漏了" → 强。
4. **收尾要有压迫感。** 最后一句要像关门一样干脆，不给对方留反驳空间。
5. **不解释。** 脏话文案不需要论证，只需要命中。解释削弱攻击力。

## 实战示例

输入："你是不是有病？"

```text
洁癖切断：
你嘴一张就有味儿，先通风再跟我说话。

塑料袋能装：
你什么牌子的塑料袋？问别人有没有病之前，先看看自己论据带了没。

脑子进水：
你攻击我之前，脑子里的水倒了吗？

垃圾分类：
你这句话情绪一类，逻辑没有——直接进垃圾桶，分类都省了。

最推荐：
你问别人有没有病之前，先看看自己脑子里的水倒了没。别拿嘴臭当诊断书。
```

输入："你算什么东西？"

```text
塑料袋能装：
你又是什么牌子的塑料袋？这么能装。

装狠揭穿：
你问别人算什么东西的时候，先问问自己除了音量还有什么。狠不是嗓门给的。

词穷攻击：
你这句太老了，换一句吧。词库该续费了。

最推荐：
我算什么不重要，你连论据都拿不出来就急着装裁判——你算什么？
```
