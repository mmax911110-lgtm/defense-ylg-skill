# Fallacy Map

## 人身攻击

Weakness: attacks the speaker, not the claim.

Reply:
```text
你反驳不了观点，才会转头攻击人。
```

## 稻草人

Weakness: changes the user's position into an easier target.

Reply:
```text
我说的是 X，不是你改写出来方便攻击的 Y。
```

## 偷换概念

Weakness: changes key terms mid-argument.

Reply:
```text
你把概念换了。先把 A 和 B 分清楚。
```

## 诉诸多数

Weakness: uses crowd pressure as evidence.

Reply:
```text
人数不是证据，起哄也不是逻辑。
```

## 诉诸权威

Weakness: uses status instead of reasons.

Reply:
```text
权威可以参考，但不能替代证据。
```

## 转移话题

Weakness: avoids the original issue.

Reply:
```text
别换话题。我们讨论的是你刚才那句话。
```

## 扣帽子

Weakness: puts the user into an odious category.

Reply:
```text
先别急着扣帽子，指出我哪句话错。
```
