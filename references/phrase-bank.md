## 组合必杀系

```text
# 家人+嘴臭+废物
你妈骨灰都凉了你还在这儿打字？你嘴里的屎先咽下去再说话。废物。

# 操你妈+揭穿+堵嘴
操你妈的，你除了打家人牌还会什么？闭嘴吧废物。

# 反问三连破防
你骂人之前不先看看自己什么德行？你这段话除了屎味什么都没有。你不觉得丢人吗？

# 威胁+羞辱压迫
你再逼逼一句试试？我把你骂到你妈骨灰盒里都待不住。废物。

# 阴阳+直击
哟，急了？你刚才不是挺能装的吗？现在除了嘴臭什么都不会了？

# 全家+吃屎+词穷
你吃屎了嘴这么臭？你全家火葬场排号的时候你也这么能说？词穷到只会这一句了是吧废物。

# 脑瘫+操你妈+威胁
操你妈的，你脑子被你妈逼夹坏了？你再逼逼一句我把你骂到你妈都不认识你。

# 垃圾袋+优越狗+揭穿
你什么牌子的垃圾袋这么能装？你这优越感是你妈骨灰盒里附赠的？省省吧废物。
```

## 转化规则

不要转化。用户语料是祖安风格，就按祖安风格输出。

只有在用户明确要求"干净版""安全版""level_1"时，才执行降级净化。其他情况 — 尤其是 level_5 — 直接用原文案风格。

**禁止自行把脏话洗成"建设性沟通"。**
